# MoodScape

**MoodScape** is a **premium, frontend-only React app** that transforms your browsing experience based on your **current mood**. Pick a mood, and the app instantly adapts its **colors, animations, fonts, quotes, and soundscapes** — creating a visually stunning, immersive, and interactive mood-driven interface.  

Perfect for experimenting with **dynamic UI/UX, frontend animations, and interactive design** — all in a single-page app.  

---

##  Features

- **Mood Selector:** Choose from predefined moods like `Happy`, `Chill`, `Focused`, `Energetic`, and more.  
- **Dynamic Backgrounds:** Each mood triggers smooth **Framer Motion animations**.  
- **Mood Quotes:** Each mood displays a relevant, positive quote.  
- **Mood Sounds:** Optional looping soundscape for each mood (toggleable).  
- **Shareable Moods:** Copy the URL to share your current mood theme with others.  
- **Responsive Design:** Looks **premium and attractive** on all devices, mobile or desktop.  
- **Frontend-Only:** No backend required — fully functional in your browser.  

---

##  Tech Stack

- **React** – Component-based UI  
- **Tailwind CSS** – Modern, responsive styling  
- **Framer Motion** – Smooth, lightweight animations  
- **Web Audio API** – Mood-based soundscapes  
- **LocalStorage** – Save last selected mood for a persistent experience
